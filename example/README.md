# Glint showcase

An executable proof that Glint can render, skin and animate characters, run
full angular rigid-body physics through the optional Box3D backend, drive a
four-wheel arcade vehicle, simulate/render pooled particles, compile typed
shader graphs, play spatial audio, receive gestures, hot reload, and sit under
ordinary Flutter UI. The launcher contains seven focused demos, including
interactive crossfades between the Fox's Survey, Walk, and Run clips.

```sh
flutter run -d macos
```

The advanced systems lab uses the optional native audio backend. Install CMake
before its first desktop build (`brew install cmake` on macOS).
